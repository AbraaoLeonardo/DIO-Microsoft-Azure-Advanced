# DIO-Microsoft-Azure-Advanced
Projeto baseados no treinamento da DIO para a certificação Azure Associate.

## Sobre o que é o projeto?
O projeto é uma TODO feita com react e golang.

## Ferramentas utilizadas
- Golang - backend
- React - Frontend
- Postgres - Banco de dados
- Docker - Gerar as Imagens
- Kubernetes - Orquestração de Containers
- Google Cloud Plataform - Infraestrutura
- Terraform - Criação da Infraestrutura
